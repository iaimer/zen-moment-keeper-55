# ZenJournal — 安卓原生风格日记 App

## 概述

一款面向移动端的全屏日记应用，作为 Obsidian 的交互前端，专注于高效录入、数据量化和结构化 Markdown 导出。使用 IndexedDB 离线存储，预留 Capacitor 打包接口。

---

## 1. 设计系统与全局布局

- **Material Design 3 风格**：大圆角卡片（16-20px）、奶油色暖色系主题配色
- **底部导航栏**：两个 Tab —「今日记录」和「照片日历」，带图标和标签
- **沉浸式状态栏**适配，安全区域 padding
- **全局设置入口**（右上角齿轮图标）

## 2. 今日记录页（主页）

从上到下依次排列以下模块：

### 2a. 混合型习惯追踪卡片（顶部）

- **量化项**：饮水（数字输入/ml）、步数（数字输入/步），带 +/- 快捷按钮
- **待办项**：复选框列表（学语言、阅读、早起等），可在设置中增删自定义项

### 2b. 每日小确幸卡片

- 特殊视觉样式（渐变或艺术背景、粗体大字）
- 全天仅限输入一条，输入后锁定显示，可编辑
- 支持关联一张照片

### 2c. 时间轴记录

- 流式卡片布局，左侧时间线装饰
- 每条记录：自动时间戳 + 文本 + 可选图片
- 底部浮动按钮添加新记录
- 图片存储至 IndexedDB

### 2d. 复盘与寄语区

- 「觉察和反思」文本区域
- 「明日寄语」单行输入框

## 3. 照片日历视图页

- 月视图日历网格
- 有照片的日期格子显示小确幸照片作为背景缩略图
- 点击日期弹出浮窗，预览当天的小确幸文字和照片
- 月份切换导航

## 4. 设置页

- 管理自定义习惯项（增加/删除待办习惯）
- Obsidian 导出功能入口
- 数据管理（清除/备份）

## 5. 数据层（IndexedDB）

- 使用 IndexedDB 存储所有日记数据和图片 Blob
- 按日期索引的数据结构：习惯数据、时间轴条目、小确幸、复盘文本
- 图片单独存储，通过 ID 关联

## 6. Obsidian Markdown 导出

- 生成 `YYYY-MM-DD.md` 文件，严格按照指定格式排列：
  - 习惯追踪（Markdown 复选框）
  - 🕰️ 时间轴
  - ✨ 每日小确幸
  - 🧠 觉察与反思
  - 📝 明日寄语
- 图片以 `![[Pasted image X.jpg]]` 格式引用，导出时提示用户图片存储路径
- 支持单日导出和批量导出，下载为 `.md` 文件

## 7. Capacitor 预留

- 配置 Capacitor 初始化，便于后续打包为 `.apk`
